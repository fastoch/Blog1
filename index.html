My journey

 I've been an agricultural engineer, then a teacher, before finally finding what really drives me: Computer Science.
 My first four years in this field were spent mostly as a member of an IT support team. Which taught me a lot, but didn't satisfy my need for learning.
 Hence, I decided to become a web developer and started to learn Front-End Web Development.

 But you can't really become a web developer on your own...
 I coudn't have made it without the help of people who have the kindness to share their knowledge freely and helped me acquire the skills I needed.
 Which is why I created this blog, to bring my humble contribution to this amazing community of programmers by returning the favor I benefited.

What you need to learn as a front-end web developer

A front-end developer must master the following languages:

  • HTML
  • CSS
  • JavaScript

You won't actually use JavaScript as such, but probably one of these popular JavaScript frameworks:
  • Angular
  • React
  • Vue
To know more about frameworks.

As a front-end web developer, you will also need to familiarize yourself with some code editor.
Many developers use Visual Studio Code, I'm one of those. But you have many other options at your disposal.
